# 💳 Credit Risk Analysis & Prediction

## 📊 Project Overview
This project analyzes loan data to identify risk patterns and predict loan defaults using Data Analytics and Machine Learning techniques.

---

## ⚙️ Tech Stack
- Python (Pandas, Scikit-learn)
- SQL
- Power BI

---

## 🔍 Project Phases

### Phase 1: Data Understanding (Python EDA)
- Data cleaning & profiling
- Missing value analysis
- Distribution visualization

### Phase 2: Data Analysis (SQL + Python)
- Loan segmentation
- Risk analysis by grade, purpose
- Feature engineering

### Phase 3: Machine Learning
- Logistic Regression model
- Default prediction
- Accuracy evaluation

---

## 🤖 ML Model
- Model: Logistic Regression
- Features:
  - Loan Amount
  - Interest Rate
  - DTI
- Accuracy: XX%

---

## 📊 Dashboard
Power BI dashboard shows:
- Good vs Bad loans
- Risk segmentation
- KPI trends

---

## 🚀 Key Insights
- High DTI = Higher default risk
- Grade G loans are risky
- Debt consolidation loans show higher defaults
